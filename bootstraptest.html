Project Overlord

  • Project Overlord was Aurora High School's robotics team entry for the Robofest exhibition games.
  • The Project was to build a unique, versatile, autonomous quadcopter.
  • Team Captain, the team won second place at the Robofest World championship 2012, in the respective division.
  • Designed, programmed and implemented a custom quadcopter design (software, control and mechanical). The software was a real-time multithreaded C++ application.
  • Ported and integrated an open-source library for communications with an existing ground control station via TCP/IP over WiFi.
  • Derived linearly independent control vectors, created multiple mechanical rigs for tuning those vectors.
  • Soldered and assembled the components.
  • Robofest 2012 competition Video submission

Motion Detection Suite

  • The project was done for the final project of grade 12 Computer Science.
  • The program was written in C++.
  • The final project was coded in collaboration with Jack Mosely.
  • Created and implemented software to analyze a series of frames to detect and track moving objects.
  • Optimized algorithms for motion detection and noise reduction.
  • The github for the project can be found here.
